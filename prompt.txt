Car Rental Website Project - Complete Overview
Project Description
I'm building a car rental website called "Sihabi Cars" (Wheelie Happy Car) - a full-stack web application for a Moroccan car rental company. The website allows customers to browse cars, make bookings, and manage their reservations. It includes an admin dashboard for managing cars, bookings, users, and messages.

Tech Stack
Frontend : React + TypeScript + Vite

UI Framework : Tailwind CSS + shadcn/ui components

Backend : Node.js + Express

Database : Supabase (PostgreSQL)

State Management : TanStack Query (React Query)

Routing : React Router

Animations : Framer Motion

Forms : React Hook Form

Notifications : Sonner (toast notifications)

Icons : Lucide React

Key Features
1. Multi-language Support (i18n)
Languages: French (default), Arabic (RTL support), English

Implementation: Custom useLanguage hook with translation keys

File: src/hooks/use-language.tsx

All UI text uses translation keys like t('nav.home'), t('cars.bookNow'), etc.

Language preference stored in localStorage

Arabic language automatically enables RTL (right-to-left) layout

2. Authentication System
User registration and login

Role-based access (user/admin)

Protected routes

User profile management

File: src/hooks/use-auth.tsx

3. Car Management
Browse available cars

Filter by category

Search by name/brand

Car details page with image gallery

Admin can add/edit/delete cars

Multiple images per car with gallery view

Car specifications (seats, transmission, fuel type, etc.)

4. Booking System
Date range selection for rentals

Price calculation based on rental days

Payment method selection (cash/card)

Booking status workflow:

Pending: Initial booking, user can cancel

Confirmed: Admin confirmed, user must contact admin to cancel

Completed: Rental finished

Cancelled: Booking cancelled

My Bookings page for users to view their reservations

Current file: src/pages/MyBookingsPage.tsx

5. Admin Dashboard
Analytics (revenue tracking: daily, monthly, yearly, total)

Car management (CRUD operations)

Booking management (view, update status, delete)

User management (view users, edit roles)

Contact messages management

Search and filter functionality

6. Contact System
Contact form with email integration

WhatsApp integration for direct messaging

Business hours and location information

Google Maps integration

7. UI Components
Responsive design (mobile-first)

Glass morphism effects

Smooth animations with Framer Motion

Toast notifications for user feedback

Modal dialogs for confirmations

Badge components for status display

Database Schema (Supabase)
Tables:
users - User accounts (id, email, password, first_name, last_name, phone, city, role)

cars - Car inventory (id, brand, name, category, price_per_day, seats, transmission, fuel_type, description, is_available, is_featured)

car_images - Car photos (id, car_id, image_url, display_order)

bookings - Rental bookings (id, car_id, user_id, customer_name, customer_email, customer_phone, start_date, end_date, total_price, payment_method, status, created_at)

contact_messages - Contact form submissions
